# Dojo Helper



## Install

first install the node dependencies
```
$ npm install
```

install the electron-prebuilt
```
$ npm install electron-prebuilt -g
```

### Run

```
$ npm start
```

### Build

'll be created a folder with DojoHelper.app name 

```
$ npm run build
```
